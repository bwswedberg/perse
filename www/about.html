Tech

PerSE was built with several javascript libraries and toolkits.

  • d3 for visualization support
  • jquery for being awesome
  • jquery calendars (the Keith Wood plugin) for calendar support
  • bootstrap for layout support
  • papaparse for file parsing
  • crossfilter for data mngmt
  • metricsgraphics for charting support
  • openlayers 3 for mapping framework
  • openstreetmap for map tiles
  • require for dependency mngmt

Other notable assistance came from Cindy Brewer's ColorBrewer palette and the a helpful js port via d3.
